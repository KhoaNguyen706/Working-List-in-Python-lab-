List Operations Quick Reference

  1. .append(item) - Adds an item to the end of the list
  2. .pop() - Removes and returns the last item in the list
  3. .pop(index) - Removes and returns the item at the specified index
  4. .index(item) - Returns the index of the first occurrence of the item
  5. list[index] - Accesses the element at the specified index (0-based)
  6. list[-1] - Accesses the last element, -2 for second-to-last, etc.
  7. .sort() - Sorts the list in-place (modifies the original list)
  8. sorted(list) - Returns a new sorted list without modifying the original
  9. .reverse() - Reverses the order of items in the list
  10. len(list) - Returns the number of items in the list

Slicing List Elements


# Slicing lists is used to easily access to specific List
# list_name[start:end:step] 
# start : index to begin the slice (inclusive)
# end : Index to end the slice (exclusive)
# step : Step size, specifying the interval between elements
fruits = ["apple", "banana", "cherry", "date", "elderberry"]

# Slicing lists
print(fruits[1:3])     # Elements from index 1 to 2
print(fruits[:3])      # Elements from start to index 2
print(fruits[2:])      # Elements from index 2 to end
print(fruits[:])       # All elements (creates a copy)

Output:


['banana', 'cherry']
['apple', 'banana', 'cherry']
['cherry', 'date', 'elderberry']
['apple', 'banana', 'cherry', 'date', 'elderberry']

Modifying Lists


fruits = ["apple", "banana", "cherry"]
print("Original list:", fruits)

# Changing an element
fruits[1] = "blueberry"
print("After changing element:", fruits)

# Adding elements
fruits.append("date")
print("After append:", fruits)

fruits.insert(2, "grape")
print("After insert:", fruits)

fruits.extend(["kiwi", "lemon"])
print("After extend:", fruits)

# Removing elements
fruits.remove("apple")
print("After remove:", fruits)

popped = fruits.pop()
print("Popped item:", popped)
print("After pop:", fruits)

del fruits[0]
print("After del:", fruits)

# Clearing the list
fruits.clear()
print("After clear:", fruits)

Output:


Original list: ['apple', 'banana', 'cherry']
After changing element: ['apple', 'blueberry', 'cherry']
After append: ['apple', 'blueberry', 'cherry', 'date']
After insert: ['apple', 'blueberry', 'grape', 'cherry', 'date']
After extend: ['apple', 'blueberry', 'grape', 'cherry', 'date', 'kiwi', 'lemon']
After remove: ['blueberry', 'grape', 'cherry', 'date', 'kiwi', 'lemon']
Popped item: lemon
After pop: ['blueberry', 'grape', 'cherry', 'date', 'kiwi']
After del: ['grape', 'cherry', 'date', 'kiwi']
After clear: []

Looping through Lists

Basic For Loop

The simplest way to iterate through a list is with a for loop. This directly accesses each element:


        fruits_list = ["apple", "pineapple", "banana", "watermelon"]
        
        # Basic for loop
        print("Loop through each item:")
        for fruit in fruits_list:
            print(fruit)

Loop with Index

When you need the position of elements, you can loop through indices using the range and len functions:


        # Loop with index
        print("\nLoop with index:")
        for i in range(len(fruits_list)): 
            print(f"Index {i}: {fruits_list[i]}")

Loop with Enumerate

The enumerate function provides both the index and element in a single loop - this is often the most efficient approach:


        # Loop with enumerate
        print("\nLoop with enumerate:")
        for index, fruit in enumerate(fruits_list):
            print(f"Position {index}: {fruit}")

Output for all loops:


        Loop through each item:
        apple
        pineapple
        banana
        watermelon
        
        Loop with index:
        Index 0: apple
        Index 1: pineapple
        Index 2: banana
        Index 3: watermelon
        
        Loop with enumerate:
        Position 0: apple
        Position 1: pineapple
        Position 2: banana
        Position 3: watermelon
